# \# QuickCare 🏥

# 

# QuickCare is a clinic appointment booking and management system built with \*\*ASP.NET MVC\*\*, \*\*Dapper\*\*, and \*\*SQL Server\*\*.

# 

# It helps clinics manage doctors, patients, receptionists, and appointments through a secure role-based system.

# 

# \---

# 

# \## 🚀 Features

# 

# \### 🔐 Authentication \& Authorization

# \- Secure Login System

# \- Role-based Access Control

# \- Separate Dashboards for Receptionist and Doctor

# 

# \### 👨‍⚕️ Doctor Management

# \- Add / Update / Delete Doctors

# \- Doctor Availability Schedule

# \- Specialization \& Qualification Management

# 

# \### 🧑‍🤝‍🧑 Patient Management

# \- Register Patients

# \- Manage Patient Records

# \- Medical History \& Allergies

# 

# \### 📅 Appointment Management

# \- Book Appointments

# \- View Doctor-wise Appointments

# \- Update Appointment Status

# \- Prevent Double Booking

# 

# \### 🧾 Receptionist Panel

# \- Manage Daily Appointments

# \- Create Bookings for Patients

#
